Benefits of GOV.UK prototype kit

Using GOV.UK prototype kit you can:

  • prototype services that look and work like GOV.UK
  • test usability of your service
  • quickly iterate your service's design
  • document the development of your service

Getting started

The prototype kit includes:

  • how to install it
  • tutorials on how to use it
  • examples
  • coded design patterns
